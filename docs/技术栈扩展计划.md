#### 1. Redis 深度应用

**应用场景**：
- 帖子点赞/收藏计数缓存（String）
- 用户点赞记录防重复（Set）
- 热帖排行榜（ZSet）
- 用户签到功能（BitMap）
- 分布式锁（防止重复提交）

**代码示例**：
```java
// 1. 点赞计数
redisTemplate.opsForValue().increment("post:like:" + postId);

// 2. 用户点赞记录 - 防重复
redisTemplate.opsForSet().add("user:liked:" + userId, postId);
redisTemplate.opsForSet().isMember("user:liked:" + userId, postId);

// 3. 热帖排行榜
redisTemplate.opsForZSet().incrementScore("hot:posts:daily", postId, score);
redisTemplate.opsForZSet().reverseRangeWithScores("hot:posts:daily", 0, 9);

// 4. 分布式锁
Boolean locked = redisTemplate.opsForValue().setIfAbsent("lock:post:" + postId, "1", 10, TimeUnit.SECONDS);
```

**简历写法**：
> 深度应用 Redis 数据结构，使用 ZSet 实现热帖排行榜、Set 存储点赞记录防重复、分布式锁防止并发问题，缓存命中率 95%+

**依赖**：已有，无需添加

#### 2. RabbitMQ 消息队列 ✅ 已实现

**已实现场景**：
| 场景 | 交换机类型 | 状态 |
|------|-----------|------|
| 验证码邮件异步发送 | Direct | ✅ 已实现 |
| 审核结果邮件通知 | Direct | ✅ 已实现 |
| 点赞/评论/关注通知 | Topic | ✅ 已实现 |

**可扩展场景**：
| 场景 | 说明 | 优先级 |
|------|------|--------|
| **延迟消息** | 帖子定时发布、举报24h未处理提醒 | 🥇 高 |
| **系统公告广播** | Fanout 交换机推送全站公告 | 🥇 高 |
| **图片异步处理** | 上传后异步压缩/加水印 | 🥈 中 |
| **Excel 导出** | 大数据量异步生成，完成通知下载 | 🥈 中 |
| **积分系统** | 用户行为异步计算积分奖励 | 🥈 中 |
| **削峰填谷** | 活动抢票/课程抢选，保护数据库 | 🥉 低 |
| **敏感词审核** | 帖子先发布，异步审核过滤 | 🥉 低 |

**代码示例**：
```java
// 生产者 - 发送邮件消息
@Autowired
private RabbitTemplate rabbitTemplate;

public void sendEmailAsync(String to, String subject, String content) {
    EmailMessage message = new EmailMessage(to, subject, content);
    rabbitTemplate.convertAndSend("email.exchange", "email.routing.key", message);
}

// 消费者 - 异步处理邮件（手动ACK）
@RabbitListener(queues = "email.queue")
public void handleEmailMessage(EmailMessage message, Channel channel, Message msg) {
    try {
        emailService.sendEmail(message.getTo(), message.getSubject(), message.getContent());
        channel.basicAck(msg.getMessageProperties().getDeliveryTag(), false);
    } catch (Exception e) {
        channel.basicNack(msg.getMessageProperties().getDeliveryTag(), false, false);
    }
}
```

**简历写法**：
> 引入 RabbitMQ 实现邮件通知异步化，采用 Direct/Topic 交换机分发消息，配置手动 ACK + 消息持久化保证可靠性，接口响应时间从 2s 降至 200ms

**依赖**：
```xml
<dependency>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-starter-amqp</artifactId>
</dependency>
```

#### 3. Spring Cache 声明式缓存

**应用场景**：
- 热门帖子缓存
- 用户信息缓存
- 标签列表缓存
- 公告信息缓存

**代码示例**：
```java
@Service
public class PostServiceImpl {
    
    @Cacheable(value = "post", key = "#id", unless = "#result == null")
    public Post getPostById(Long id) {
        return postMapper.selectById(id);
    }
    
    @CachePut(value = "post", key = "#post.id")
    public Post updatePost(Post post) {
        postMapper.updateById(post);
        return post;
    }
    
    @CacheEvict(value = "post", key = "#id")
    public void deletePost(Long id) {
        postMapper.deleteById(id);
    }
    
    @Cacheable(value = "hotPosts", key = "'top10'", expire = 300)
    public List<Post> getHotPosts() {
        return postMapper.selectHotPosts(10);
    }
}
```

**简历写法**：
> 整合 Spring Cache + Redis 实现声明式缓存，热点数据自动缓存，数据库查询降低 70%

**依赖**：
```xml
<dependency>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-starter-cache</artifactId>
</dependency>
```

#### 4. Elasticsearch 全文搜索

**应用场景**：
- 帖子标题/内容全文检索
- 搜索关键词高亮
- 搜索建议/自动补全
- 相关度排序

**代码示例**：
```java
@Document(indexName = "post")
public class PostDocument {
    @Id
    private Long id;
    
    @Field(type = FieldType.Text, analyzer = "ik_max_word")
    private String title;
    
    @Field(type = FieldType.Text, analyzer = "ik_max_word")
    private String content;
    
    @Field(type = FieldType.Keyword)
    private List<String> tags;
}

// 搜索服务
public Page<PostDocument> search(String keyword, Pageable pageable) {
    NativeSearchQuery query = new NativeSearchQueryBuilder()
        .withQuery(QueryBuilders.multiMatchQuery(keyword, "title", "content"))
        .withHighlightFields(
            new HighlightBuilder.Field("title"),
            new HighlightBuilder.Field("content"))
        .withPageable(pageable)
        .build();
    return elasticsearchRestTemplate.search(query, PostDocument.class);
}
```

**简历写法**：
> 集成 Elasticsearch 实现帖子全文检索，支持 IK 中文分词、关键词高亮、相关度排序，搜索响应 < 100ms

**依赖**：
```xml
<dependency>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-starter-data-elasticsearch</artifactId>
</dependency>
```

#### 5. XXL-Job 分布式定时任务

**应用场景**：
- 每日热帖榜单统计
- 过期数据定时清理
- 用户活跃度统计
- 定时推送系统通知

**代码示例**：
```java
@Component
public class PostJobHandler {
    
    @XxlJob("calculateHotPostsJob")
    public void calculateHotPosts() {
        // 每日凌晨统计热帖
        List<Post> hotPosts = postService.calculateDailyHotPosts();
        redisTemplate.opsForValue().set("hot:posts:daily", hotPosts);
        XxlJobHelper.log("热帖统计完成，共 {} 条", hotPosts.size());
    }
    
    @XxlJob("cleanExpiredDataJob")
    public void cleanExpiredData() {
        // 清理30天前的浏览记录
        int count = viewRecordMapper.deleteExpired(30);
        XxlJobHelper.log("清理过期数据 {} 条", count);
    }
}
```

**简历写法**：
> 使用 XXL-Job 实现分布式定时任务调度，每日自动统计热帖榜单、清理过期数据，支持任务监控与失败重试

**依赖**：
```xml
<dependency>
    <groupId>com.xuxueli</groupId>
    <artifactId>xxl-job-core</artifactId>
    <version>2.4.0</version>
</dependency>
```

#### 6. Sentinel 接口限流熔断

**应用场景**：
- 接口 QPS 限流
- 热点参数限流
- 服务熔断降级
- 防止恶意刷接口

**代码示例**：
```java
@RestController
public class PostController {
    
    @GetMapping("/front/post/{id}")
    @SentinelResource(value = "getPost", 
                      blockHandler = "getPostBlockHandler",
                      fallback = "getPostFallback")
    public Result<Post> getPost(@PathVariable Long id) {
        return Result.ok(postService.getPostById(id));
    }
    
    // 限流处理
    public Result<Post> getPostBlockHandler(Long id, BlockException e) {
        return Result.fail("访问过于频繁，请稍后再试");
    }
    
    // 降级处理
    public Result<Post> getPostFallback(Long id, Throwable t) {
        return Result.fail("系统繁忙，请稍后再试");
    }
}
```

**简历写法**：
> 引入 Sentinel 实现接口限流熔断，配置 QPS 阈值与降级策略，保障系统高可用，防止恶意请求

**依赖**：
```xml
<dependency>
    <groupId>com.alibaba.cloud</groupId>
    <artifactId>spring-cloud-starter-alibaba-sentinel</artifactId>
    <version>2021.0.5.0</version>
</dependency>
```
